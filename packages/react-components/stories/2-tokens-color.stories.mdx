# Color Tokens

To be able to commuicate UI symantic in a smooth and consitent way, we group our color scheme by: `brand`, `background`, `text` and `ui` itself groups.

Color scheme is composed for both `light` and `dark` modes.


## Light
<ColorPalette>
  {palette(theme.colors).map(colorItem => (
    <ColorItem key={colorItem.title} {...colorItem} />
  ))}
</ColorPalette>


## Dark
<ColorPalette>
  {palette({...theme.colors, ...theme.colors.modes.dark}).map(colorItem => (
    <ColorItem key={colorItem.title} {...colorItem} />
  ))}
</ColorPalette>
