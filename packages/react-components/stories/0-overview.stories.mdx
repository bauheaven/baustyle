# Baustyle

<Subtitle>
  Multivariant extentable React UI Components, with focus on accebility and performance.
</Subtitle>

## What does it build on.
Library builds upon Typescript, React, [Styled System](https://styled-system.com/) and [Styled Components](https://styled-components.com/), and relies on a `theme` object
extended out of [System UI Theme Specification](https://system-ui.com/theme/) 


## Install.

you need already to have preinstalled `styled-components`, `styled-system` and optionally `@styled-system/css`

```
npm install @bauheaven/baustyle-react
```
